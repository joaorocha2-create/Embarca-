# Embarcaê

## Descrição
O projeto busca resolver a falta de transporte coletivo em Iporá, atrvés do protótipo de aplicativo de transporte público com rotas que integram a maioria dos bairros com o centro urbano, de forma que atenda a grande parte dos morradores que possuem dificuldade de se deslocarem pelo município.
